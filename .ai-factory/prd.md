# DuoTrack

## Platform
앱인토스 (Vite + React + TDS)

## One-liner
국내 어학 앱 사용자의 최대 불만인 '광고 중단 + 학습 흐름 파괴' 문제와 Duolingo식 게임화의 낮은 실력 향상 효과에 지친 학습자들을 위해, 광고 없는 집중 학습 + 실제 시험(토익·오픽) 점수 연동 트래킹으로 '학습 ROI'를 증명해주는 영어 학습 성과 관리 앱

## Problem
Duolingo는 광고가 학습 흐름을 자주 방해하고(앱스토어 1점 리뷰 다수), 게임 요소가 재미는 있으나 토익 점수 향상과 연결되지 않는다는 불만이 지배적. 국내 어학 학원·인강은 학습은 하는데 '얼마나 늘었는지' 객관적 측정이 안 됨. 취업 준비생·직장인은 토익·오픽 목표 점수까지의 진척도를 한눈에 보고 싶어함.

## Target Users
토익·오픽 준비 중인 20-30대 취업 준비생 및 직장인, Duolingo에 실망한 중급 이상 영어 학습자, 영어 학습 비용 대비 효과를 측정하고 싶은 층

## Core Features
1. 목표 시험(토익/오픽/텝스) 설정 → AI가 현재 실력 진단 후 맞춤 학습 경로 설계
2. 광고 없는 집중 학습 세션 (25분 포모도로 기반, 중간 알림 없음)
3. 모의 시험 → 실제 시험 점수 입력 → 예측 정확도 및 학습 효율 분석
4. 취약 파트 자동 감지 + 해당 파트 집중 문제 자동 생성
5. 학습 투자 대비 점수 향상 ROI 리포트 ('시간당 점수 향상 효율' 시각화)

## Monetization
- Model: freemium subscription
- Estimated MRR: $1,400
- Reasoning: 무료: 기본 진단 + 주 3회 세션. 유료(월 7,900원): 무제한 세션 + 모의 시험 + ROI 리포트. 토익 학원비 월 10-20만원 대비 압도적 가격 경쟁력. 취업 시즌(상반기 3-5월, 하반기 9-11월) 수요 집중. 200명 전환 목표 시 약 $1,200, 취업 시즌 할증 고려 $1,400 MRR 산정. 단, health 카테고리 실패 패턴 교훈으로 유료 기능 명확한 차별화 필수.
